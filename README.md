# Ludic Historian

_A Game Historian's View of Ludic Ambient Heuristics_

Running `npm start` will copy all `src` files to `public` and startsthe browser-sync server at [http://localhost:3000](http://localhost:3000).

To run style lining on the scss files, you can do one of two commands:

- `npm run lint`
- `npx stylelint "**/*.scss"`

## NPM Scripts

- `start`: serve the public environment
- `lint`": apply linting rules to Sass styles
- `delete`: remove the public environment
